# 🦷 PreDent - 치과 사전 문진 챗봇 서비스

## ✨ 프로젝트 소개

본 프로젝트는 **치과 진료 과정에서 환자의 증상을 체계적으로 분석하고, 사전 진단서를 자동 생성**하며, **멀티턴 대화 기능을 지원하는 챗봇 서비스**를 개발하는 것을 목표로 합니다.  
환자의 **VAS(Visual Analogue Scale) 통증 척도 시각화**, **가능성 있는 질환을 포함한 사전 진단서 생성** 기능을 포함하며, **Django & React 기반**으로 개발되었습니다.

## 🛠️ 기술 스택

- **Frontend:** React.js
- **Backend:** Django (Python)
- **Database:** PostgreSQL
- **Cloud & Infra:** AWS (Ubuntu 24.04 LTS)
- **Agent & ML:** LangGraph, STT(Speech-to-Text)
